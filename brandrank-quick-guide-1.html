üìã Step-by-Step Instructions

  1. Go to the BrandRank Dashboard for the customer you're analyzing
  2. Screenshot the overview page that displays all four core scores:
    • Brand Health & Trust
    • Visibility
    • Vulnerability
    • Readiness
  3. Download data from each section using the new export tabs we created:
    • Navigate to each of the four score sections
    • Use the download/export functionality for each
  4. For Sources: Download visibility data only (not all source data)
  5. Create a fresh prompt on Claude.ai (first choice, though results are comparable on ChatGPT):
    • Upload the prompt template (.md preferred) from above
    • Upload all the data files you downloaded
    • Upload your screenshot
    • Ask it to run the report
  6. Quality check (10 minutes): Review the generated report to verify all data is aligning correctly with your source files
  7. Extra quality control (optional): Run the same report on ChatGPT (or whichever tool you didn't start with) and compare results for consistency
