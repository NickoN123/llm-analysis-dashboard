Amazon Rufus Answer Index

Universal Report Generation Prompt Template
YOUR TASK: Create a comprehensive HTML report called "Amazon Rufus Answer Index: [CATEGORY NAME]" that analyzes brand visibility across 60-80 consumer queries. The report shows which brands appear in 1st, 2nd, and 3rd place when consumers ask Amazon Rufus common questions in this category.

Data Input Requirements

The user will provide a CSV or structured data file containing:

  • Consumer Question - the query asked to Rufus
  • 1st Place Brand - top-ranked product/brand
  • 2nd Place Brand - second-ranked product/brand
  • 3rd Place Brand - third-ranked product/brand
  • Explanation - why this ranking, key differentiators, ratings
  • Product Depth Factors - star ratings, review counts, key features

Report Structure

Header Section

Title: Amazon Rufus Answer Index: [CATEGORY NAME]
Subtitle: Rufus Answers to Common Queries: Powered by BrandRank / [MONTH YEAR]

Main Components (in this exact order):

1. Search & Filter Controls

  • Search box for keywords/brands
  • Company filter buttons (extract top 6-8 brands from data)
  • Clear button
  • Search results counter

2. Interactive Data Table

Sticky header with columns:

Column Name Width Styling
Customer Question 28% Standard white text
1st Place 10% GOLD (#ffd700)
2nd Place 10% SILVER (#c0c0c0)
3rd Place 10% BRONZE (#cd7f32)
Why This Ranking? 30% Standard text
Product Depth Factors 22% Small gray text
  • Category headers organizing questions into 6-10 logical groups
  • Hover tooltips on brand names showing full competitive profiles

3. Executive Summary

Two-column grid:

  • LEFT: "✓ Dominant Categories" (green theme #4CAF50) - where focal brand leads
  • RIGHT: "⚠ Vulnerable Categories" (red theme #f44336) - where focal brand struggles

Strategic insights section analyzing competitive threats with data-driven insights and specific numbers (e.g., "10 of 17 #1 positions = 59% dominance")

4. Competitive Landscape

  • 2-column grid of top 8-10 competitor profiles
  • Each profile: Brand name, 50-75 word description
  • Include: positioning, star ratings, review counts, key differentiators

5. About BrandRank

At the very bottom, include:

BrandRank works with over 60 global brands to strengthen trust and purchase intent by uncovering vulnerabilities, purchase-driving insights, and growth opportunities across AI answer engines. As a SaaS platform, we track all major answer engines to improve visibility, ensure content is AI-ready, and rigorously measure brand claims and instructional accuracy.

  • Gradient backgrounds on summary sections

JavaScript Functionality

  1. addBrandTooltips() - Wraps brand names with hover tooltips containing competitive profiles

Category Organization Logic

Automatically group the 60-80 questions into 6-10 logical categories based on:

  • Pain points addressed (e.g., headaches, body pain, fever)
  • User demographics (e.g., children's, adults, seniors, pregnancy)
  • Product attributes (e.g., easy-to-swallow, clean label, fast-acting)
  • Special conditions (e.g., sensitive users, dietary restrictions)

Each category should contain 5-15 related questions.

Executive Summary Generation

Dominant Categories Section:

  • Calculate % of #1 positions for focal brand in each category
  • Highlight categories where focal brand achieves 50%+ dominance
  • Include specific product examples with star ratings and review counts
  • Identify unique innovations or differentiators

Vulnerable Categories Section:

  • Identify categories where focal brand has 0-2 #1 positions
  • Name the dominant competitor in each vulnerable category
  • Explain their competitive advantage (price, formulation, positioning)

Strategic Insights:

Summarize in 2-3 paragraphs:

  1. Overall competitive position
  2. 3-5 distinct competitive threats with specifics
  3. Critical opportunities based on strengths

Competitive Landscape Profiles

For each major competitor (top 8-10 brands by appearance frequency):

  • Extract positioning claims from explanations
  • Aggregate star ratings and review volumes
  • Identify key differentiators and innovations
  • Summarize in 50-75 words
  • Assign visual hierarchy: Gold (1st), Silver (2nd), Bronze (3rd), Standard for others

Output File Naming

Save as: [brand-name]-rufus-analysis.html

Example: tylenol-rufus-analysis.html

Critical Requirements

  1. Zero Redundancy: Each question appears exactly once in the table
  2. Data Integrity: Every row must have all 6 columns populated from source data
  3. Tooltip Accuracy: Brand profiles must match competitive landscape descriptions exactly
  4. Responsive Design: Table must scroll independently from rest of page
  5. Professional Polish: No lorem ipsum, no placeholder text, no broken functionality
